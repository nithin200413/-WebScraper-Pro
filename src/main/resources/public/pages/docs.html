How to use WebScraper Pro

Getting started

  • Create an account on the Sign Up page.
  • Sign in and you'll land on the Dashboard.
  • Paste any public URL and click Scrape.

What gets extracted

  • Meta — title, description, keywords, language, favicon, Open Graph image.
  • Links — deduplicated, classified internal vs external.
  • Images — deduplicated with previews.
  • Headings — H1–H6 outline.
  • Tables — normalized rows/columns, exportable to CSV.
  • Contacts — emails and phone numbers.
  • Text stats — word count, paragraphs, reading time.
  • Screenshot — a captured image of the page.

Exporting

Hover the Export button on the dashboard to download your results as JSON, CSV, PDF, Word (.docx), HTML source or plain text. PDF and Word files are generated on the server using OpenPDF and Apache POI.

Tech stack

Java 17 · Spark · JSoup · MySQL (XAMPP) · HikariCP · BCrypt · OpenPDF · Apache POI. The frontend is vanilla HTML/CSS/JS split into modules.
